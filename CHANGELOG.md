# Changelog

## [2.0.1] — 2026-06-14

### 修正
- フローチャートでポートをドラッグしてエッジを接続できない問題を修正
  - ポートがノードの上に重なるため、カーソルがポートに乗るとノードの `mouseleave` が発火し、ポートが即座に消えて掴めなかった。ノード／ポートのホバーを一体として扱い、離脱時は遅延して隠すよう変更

## [2.0.0] — 2026-06-14

### 追加
- フローチャート: ノードシェイプ変更（矩形 / 角丸 / ひし形 / スタジアム / 円）
- フローチャート: エッジタイプ変更（実線・点線・太線、矢印有無）
- SVG / PNG エクスポート（ツールバー・右クリックメニュー）
- ダイアグラム種別を判別できないファイル向けの選択画面
- パーサ / シリアライザのユニットテスト（`npm test`）

### 変更
- 要求仕様書をv2.0へ更新し、実装との乖離（メッセージプロトコル・データモデル・ファイル構成）を解消
- Gantt / Flowchart の WebviewPanel 管理を共通 `editorPanel.ts` に統合し、未使用の `ganttPanel.ts` / `flowchartPanel.ts` / `mermaidPanel.ts` を削除

### 修正
- フローチャートパーサがノードシェイプを解析するよう修正（型の不整合を解消）
- サンプル `sample/project.md` の破損を修復
- CRLF 環境で Mermaid フェンスが保持されない問題を修正
