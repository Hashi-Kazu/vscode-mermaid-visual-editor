# Changelog

## [1.5.7] — 2026-06-14

### 修正
- フローチャートエディタでラベル編集・エラー表示・エッジドラッグが動作しないバグを修正 (#3)
  - `_buildHtml()` に `#error-panel`・`#fc-edit-overlay`・`#fc-edit-input`・`#drag-edge-svg`・`#drag-edge-line` を追加

## [1.2.0] — 2026-06-12

### 変更
- パネル起動時・コード更新時・リセットボタン押下時のスクロール初期位置を「今日 − 3日」が左端に見える位置に変更

## [1.1.0] — 2026-06-12

### 追加
- タスク依存関係 (`after <id>`) のビジュアル設定・解除（右クリックメニュー）
- 依存関係の矢印オーバーレイ表示（SVG ベジェ曲線）
- セクション単位の折りたたみ / 展開（`▼/▶` ボタン）
- `Ctrl` + マウスホイールによるタイムラインズーム（カーソル位置を中心）
- マーメイドテールのアイコンに更新
- README および CHANGELOG を追加

### 変更
- 通常のマウスホイールはタイムラインの上下スクロールに変更（旧：ズーム）
- 依存関係を持つタスクをドラッグ移動すると依存関係が解除され絶対日付に変換

## [0.9.0] — 2026-06-11

### 追加
- ラベル列のタスク名ダブルクリックによる編集開始

### 修正
- ドラッグ閾値超過かつ正味変化ゼロのとき `render()` を呼ばないよう修正（ダブルクリックが誤動作する場合があったバグ）

## [0.8.0] — 2026-06-11

### 追加
