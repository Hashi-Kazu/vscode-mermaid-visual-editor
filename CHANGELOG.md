# Changelog

## [2.15.0] - 2026-07-13

### Changed
- ガントチャートの初期表示サイズを従来比1.2倍に拡大し、これを100%として扱うよう変更（ズーム率表示・刻み・リセット挙動は従来どおり）

## [2.14.0] - 2026-07-13

### Fixed
- ガントチャートで `duration` が短いタスクバーが実期間より広く描画され期間範囲からはみ出す不具合を修正。狭いバーではステータスインジケーターを省略し、バーを実期間幅にフィットさせるようにした（表示のみの変更で duration 等のデータは不変）

## [2.13.0] - 2026-07-09

### Added
- ガントチャートのタスクラベル列に、除外日を除いた実働日数（工数）を `(N日)` 形式で常時表示するよう追加

## [2.12.1] - 2026-07-09

### Changed
- ガントチャートのタスク名がバーに収まらない場合、省略表示ではなくバー右側に表示するよう変更
- マイルストーンのラベルを常にダイヤモンド右側に表示するよう変更
- 休日・除外日のグレーアウトをヘッダーだけでなく、セクション/タスク行のタイムライン領域にも拡張

## [2.11.1] - 2026-07-09

### Fixed
- ガントチャートの未着手・マイルストーンの紫色が濃すぎたため明るく調整
- タイムラインヘッダーの日付目盛りを中央揃えから左寄せに変更

## [2.11.0] - 2026-07-09

### Changed
- ガントチャートのステータス配色を刷新し、未着手・進行中・完了を識別しやすい配色へ変更
- 状態ピッカーと関連ドキュメントの表記を「未設定」から「未着手」へ変更
- ガントチャートの日付目盛りを月初と月曜日の週次表示へ変更
- 今日線を太くし、視認性を向上

### Added
- ガントチャートのラベル列をドラッグでリサイズできる機能を追加

## [2.10.6] - 2026-07-08

### Fixed
- ライトテーマで進行中（active）タスクバーの色が半透明に見えて視認しづらい問題を修正

## [2.10.5] - 2026-07-04

### Fixed
- 複数の Mermaid ブロックを含む Markdown で、対象外の本文やブロックが編集時に失われる問題を修正
- フローチャートの操作を連続実行した際に先行する編集が失われる問題を修正
- VS Code のテーマ切替をフローチャートへ即時反映
- ガントのセクション削除・並び替え後も折りたたみ状態と選択位置を維持

## [2.9.0] - 2026-06-26

### Changed
- Marketplace 再公開用のバージョン更新。v2.8.0 のフォーカス復元変更は revert 済みで挙動変更なし。

## [2.2.1] — 2026-06-15

### ドキュメント
- README をフローチャートエディタの機能（ノード／エッジ編集・形状・線種・既定エッジ・エクスポート・リセット等）を含む内容へ全面更新
- CHANGELOG に 2.0.2〜2.2.0 の変更履歴を追記

## [2.2.0] — 2026-06-15

### 追加
- フローチャート: 新規エッジ（ポートドラッグ追加）の既定線種をツールバー「既定エッジ」で設定可能に。MDには保存できないため、ビューア表示中のみ保持する一時設定（パネル再表示はまたいで維持、閉じて開き直すと既定の実線矢印に戻る）

## [2.1.0] – [2.1.8] — 2026-06-15

### 追加・変更
- MD記述の更新ポリシーを「遅延分離」に統一: ノードは単独宣言行・エッジはID参照のみとし、ノード編集時に対象ノードをインライン定義から集約。挿入テンプレート・サンプルも分離形式へ
- エッジの当たり判定を実線幅より広く取り、選択・編集しやすく
- ノード／ラベル編集のインライン入力欄をズーム倍率に追従（過大表示も抑制）。入力欄外クリック・パン・ズーム・リセットの直前に編集を確定
- リセット（全体表示）を実描画ジオメトリ基準の正確なセンタリング＋拡大率向上に変更
- エクスポートから編集UI由来要素（当たり判定パス・選択/ホバー装飾）を除去

### 修正
- Mermaid 11 でエッジのラベル編集・スタイル変更・削除が効かない問題（`LS-`/`LE-` クラスの固定値化に対応し `id`／`data-id` で解決）
- エッジ操作が向きを無視して逆向き・無関係エッジを取り違える問題
- ホバー時のポートが再描画・パン・ズームで画面に取り残される問題、ポートと図形端点のズレ
- 再レンダリング時に陳腐化した編集オーバーレイの破棄、未使用コード削除

## [2.0.2] – [2.0.5] — 2026-06-15

### 変更
- フローチャートの「全体表示」を「リセット」へ改称し、全ノード・エッジが収まる最大拡大＋センタリング動作に。ズーム上限を引き上げ
- エッジラベルの文字を鮮明化（合成レイヤー昇格を避け再ラスタライズ）
- ノード形状・エッジ線種の判別性を向上（輪郭の太線化・線種パターン強調）

## [2.0.1] — 2026-06-14

### 修正
- フローチャートでポートをドラッグしてエッジを接続できない問題を修正
  - ポートがノードの上に重なるため、カーソルがポートに乗るとノードの `mouseleave` が発火し、ポートが即座に消えて掴めなかった。ノード／ポートのホバーを一体として扱い、離脱時は遅延して隠すよう変更
